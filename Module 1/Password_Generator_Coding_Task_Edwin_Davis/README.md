# Password Generator

A simple, beginner-friendly command-line password generator built with Node.js that creates secure passwords with customizable options.

## Features

- Generate passwords with customizable length (8-128 characters)
- Include/exclude character types:
  - Lowercase letters (a-z)
  - Uppercase letters (A-Z)
  - Numbers (0-9)
  - Symbols (!@#$%^&*)
- Password strength indicator (Weak/Medium/Strong)
- CLI interface
- Input validation

## Prerequisites

- Node.js (version 12 or higher)

## Installation

1. Clone or download this project
2. Navigate to the project directory
3. No additional dependencies required (uses built-in Node.js modules)

## Usage

Run the password generator:

```
node index.js
```
